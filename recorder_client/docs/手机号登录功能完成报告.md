## 测试用例

### 密码登录测试
1. 输入有效手机号和密码 → 登录成功
2. 输入无效手机号 → 显示错误提示
3. 输入错误密码 → 显示错误提示
4. 多次输入错误 → 触发锁定机制

### 验证码登录测试
1. 输入有效手机号 → 发送验证码按钮可用
2. 点击发送验证码 → 启动60秒倒计时
3. 输入正确验证码 → 登录成功
4. 输入错误验证码 → 显示错误提示

### UI交互测试
1. 切换登录方式 → UI正确切换
2. 输入框焦点管理 → 错误提示清除
3. 按钮状态管理 → 根据输入动态更新

## API集成

登录功能已集成以下API：
- `AuthAPI.loginByPassword()` - 密码登录
- `AuthAPI.loginByPhone()` - 验证码登录
- `AuthAPI.getSmsCode()` - 发送验证码
- `AuthAPI.loginByWechat()` - 微信登录
