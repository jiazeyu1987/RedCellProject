## 组件使用示例

### 卡片组件
```json
{
  "usingComponents": {
    "custom-card": "/components/card/card"
  }
}
```

```html
<custom-card 
  title="标题" 
  subtitle="副标题"
  type="default"
  bind:tap="onCardTap">
  <view slot="content">自定义内容</view>
</custom-card>
```

### 按钮组件
```html
<custom-button 
  type="primary" 
  size="large"
  disabled="{{false}}"
  bind:tap="onButtonTap">
  确认
</custom-button>
```
